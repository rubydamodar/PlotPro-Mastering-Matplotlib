## **1. Line Plot** 📈

A **Line Plot** is the simplest and most fundamental plot type. It’s like the **backbone** of data visualization—simple yet powerful. It shows the relationship between two continuous variables.

### **When to use it**:

- To visualize trends or changes over time (e.g., temperature variations across days or stock market fluctuations).

### **Example**:

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [10, 20, 25, 30, 40]

plt.plot(x, y, color='blue', linestyle='--', marker='o')
plt.title('Simple Line Plot')
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.show()
```

### **Customization**:

- **Color**: You can change the color to make it more appealing.
- **Line Style**: Solid, dashed, dotted, or any other fancy line you prefer.
- **Markers**: Add markers to highlight individual data points.

---

## **2. Bar Chart** 📊

A **Bar Chart** is great for comparing categories. It's like a **full thali**, giving you a wholesome view of categorical data. The length of each bar represents the magnitude of the data point.

### **When to use it**:

- Comparing sales of different products, student marks across subjects, or population across regions.

### **Example**:

```python
categories = ['A', 'B', 'C', 'D']
values = [10, 25, 17, 35]

plt.bar(categories, values, color='orange')
plt.title('Bar Chart Example')
plt.show()
```

### **Customization**:

- Use **stacked** or **grouped bars** for comparing multiple categories.
- **Horizontal bars** for a fresh, innovative look!

---

## **3. Histogram** 📊

A **Histogram** is like a **filter coffee**—it reveals the underlying distribution of data. It groups continuous data into bins and shows how frequently each bin occurs.

### **When to use it**:

- To explore data distribution, like age groups in a population or frequency of scores in an exam.

### **Example**:

```python
import numpy as np

data = np.random.randn(1000)
plt.hist(data, bins=30, color='green', edgecolor='black')
plt.title('Histogram Example')
plt.show()
```

### **Customization**:

- **Bins**: Increase or decrease the number of bins for more or less granularity.
- Display a **cumulative** histogram to track totals.

---

## **4. Scatter Plot** ⚪

A **Scatter Plot** helps to visualize relationships between two variables by plotting points. It’s like arranging a **rangoli**—every point adds value to the overall picture.

### **When to use it**:

- Visualize correlations, like price vs. demand or age vs. height.

### **Example**:

```python
plt.scatter(x, y, color='purple', s=100)
plt.title('Scatter Plot Example')
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.show()
```

### **Customization**:

- Vary the **size** and **color** of points to represent another dimension, like population size.

---

## **5. Pie Chart** 🍰

A **Pie Chart** breaks data into proportions, just like slicing a **gulab jamun**. Each slice of the pie represents a percentage of the whole.

### **When to use it**:

- When you want to visualize proportions or parts of a whole, like market shares or survey results.

### **Example**:

```python
sizes = [30, 20, 25, 25]
labels = ['Product A', 'Product B', 'Product C', 'Product D']

plt.pie(sizes, labels=labels, autopct='%1.1f%%', startangle=140)
plt.title('Pie Chart Example')
plt.show()
```

### **Customization**:

- **Explode** a slice to emphasize it!
- Add shadows for a more 3D look.

---

## **6. Box Plot** 📦

A **Box Plot** shows data distribution using quartiles, helping you find outliers. Think of it like opening a **treasure chest**—you see the full spread of the data in one glance.

### **When to use it**:

- Comparing data distributions, especially for groups.

### **Example**:

```python
data = [np.random.randn(100) for _ in range(4)]
plt.boxplot(data, notch=True)
plt.title('Box Plot Example')
plt.show()
```

### **Customization**:

- Add **notches** for more detail about the median.
- Change colors and positions for multiple box plots.

---

## **7. Violin Plot** 🎻

A **Violin Plot** combines a box plot and a density plot, giving you insights into the distribution of data. It’s like adding a **spicy twist** to your box plot.

### **When to use it**:

- To visualize distributions in more detail, especially when comparing multiple categories.

### **Example**:

```python
import seaborn as sns

sns.violinplot(x="category", y="value", data=data)
plt.title('Violin Plot Example')
plt.show()
```

---

## **8. Area Plot** 🗻

An **Area Plot** is like layering a **delicious lasagna**—it shows how different parts contribute to a whole over time. Similar to a line plot, but with filled areas.

### **When to use it**:

- Best for visualizing stacked data over time, like revenue from multiple sources.

### **Example**:

```python
plt.fill_between(x, y1, y2, color='skyblue', alpha=0.5)
plt.title('Area Plot Example')
plt.show()
```

---

## **9. 3D Plot** 🌍

A **3D Plot** adds depth to your data, letting you visualize multiple dimensions. It’s like adding a third layer to your **favorite biryani**.

### **When to use it**:

- Ideal for visualizing geographical data or any data requiring three dimensions.

### **Example**:

```python
from mpl_toolkits.mplot3d import Axes3D

ax = plt.axes(projection='3d')
ax.plot3D(x, y, z, 'gray')
plt.title('3D Plot Example')
plt.show()
```

---

## **10. Heatmap** 🔥

A **Heatmap** uses color gradients to represent data values. It’s like spicing up your data visualization with **chutneys**—hot, spicy, and revealing.

### **When to use it**:

- Ideal for visualizing correlations, frequencies, or any tabular data.

### **Example**:

```python
sns.heatmap(data, annot=True, cmap="coolwarm")
plt.title('Heatmap Example')
plt.show()
```

---

## **11. Error Bars** 🚦

**Error Bars** are like adding caution to your plotting—letting you visualize the uncertainty or variability in your data.

### **When to use it**:

- To show variability in data, especially in scientific experiments.

### **Example**:

```python
plt.errorbar(x, y, yerr=0.2, fmt='o', ecolor='

black', capsize=5)
plt.title('Error Bars Example')
plt.show()
```
